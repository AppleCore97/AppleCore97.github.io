## 常用软件

1. 系统工具
   - Everything/ Listary    `文件搜索`
   - CCleaner      `电脑清理`
   - Bandizip      `文件压缩`
   - HoneyView     `图片查看`
   - TeamViewer/ 向日葵    `远程连接`
   - MobaXterm/ Microsoft Remote Desktop (UWP) `远程连接`
   - VMware        `虚拟机`
   - Chrome/ Microsoft Edge/ Firefox    `浏览器`

   - Snipaste       `截图贴图`
   - 快贴           `云剪贴板` <https://clipber.com/>

   - EV录屏         `录屏`

   - Microsoft Office
     - OneNote
     - OneDrive for business
     - Visio
     - islide 插件（Powerpoint)
     - Onetastic插件（OneNote）
   - Inkscape   `矢量绘图`

3. 科研
   - NoteExpress    `文献管理`
   - Mendeley       `文献管理`
   - TexLive        `LaTeX环境`
   - copytranslator `翻译软件`

4. 编程
    - Visual Studio Code
    - PyCharm
    - MATLAB

    - Keras/ PyTorch
    - Python 3.6
    - Git
